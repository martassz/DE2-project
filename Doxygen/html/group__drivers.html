◆ bme280_init()

void bme280_init ( void )

Initialize the BME280 sensor.

Performs the following:

  • Reads factory calibration parameters from the sensor's ROM.
  • Configures oversampling settings for Humidity, Temperature, and Pressure.
  • Sets the sensor mode to Normal.

◆ bme280_read()

void bme280_read ( float * temperature,
float * pressure,
float * humidity )

Read and compensate sensor data.

Reads raw ADC values from the sensor and applies the Bosch compensation formulas using the stored calibration parameters.

Parameters
[out]temperaturePointer to float variable for Temperature [°C].
[out]pressurePointer to float variable for Pressure [hPa].
