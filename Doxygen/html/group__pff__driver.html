◆ disk_initialize()

DSTATUS disk_initialize ( void )

Initialize the Disk Drive (SD Card).

  • Performs the standard SD card initialization sequence:
  1. Initialize SPI.
  2. Send dummy clocks to wake up the card.
  3. Send CMD0 to enter Idle State.
  4. Determine card type (SDv2, SDv1, or MMC) and initialize.
  5. Configure block length to 512 bytes (if not SDHC).
  6. Set SPI to double speed (f_osc/2) on success.
  • Returns
    Status code:
    Return values
    0Initialization succeeded.
    STA_NOINITInitialization failed.
    Returns
    Status code (0 = OK, STA_NOINIT = Failed).

Write partial sector to the disk.

  • Writes data to a sector on the SD card. If buff is provided, data is written. If buff is NULL and sc is non-zero, a Write Block command is initiated. If buff is NULL and sc is zero, the write transaction is finalized.
